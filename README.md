# react-native-bcrypt-cpp

Bcrypt implemented in pure C++ and linked to  React Native using Turbo Modules

## Installation

```sh
npm install react-native-bcrypt-cpp
```

## Usage


```js
import {
  generateHash,
  generateHashSync,
  validatePassword,
  validatePasswordSync,
} from 'react-native-bcrypt-cpp';

// ...

```
